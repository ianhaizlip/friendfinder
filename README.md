An app that matches people depending on a survey.

### Dependencies

* node

npm
* express
* body-parser
* path

* jQuery

### Executing program

* in terminal use node to load server.js
```
node server.js
```
* then load the local host
```
http://localhost:3000/
```
* then go to the survey
* fill out the survey
* you will be matched with people in the database
